Setup Instructions:
1. Install Python 3.12.4 from https://www.python.org/downloads/
	- Ensure that it is installed at C:\Users\%USERNAME%\AppData\Local\Programs\Python\Python[version] (this will most likely be the default)
		- %USERNAME% should be your username on your machine
	- Make sure to add Python to your Path when installing (there should be a box to check)

2. Run setup.sh file

Running Instructions:
*Note: setup instructions must be executed before program can be run*
1. After having run setup.sh, run the run.sh file
2. The script should start running and open up a terminal window, which will provide you with instructions
